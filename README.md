# Usage

## Dependencies

* JDK >= 1.8
* Maven >= 3.0

## How to package

> mvn clean package -Dmaven.test.skip=true

## Using IoTDB JDBC with Maven

```
<dependencies>
    <dependency>
      <groupId>cn.edu.tsinghua</groupId>
      <artifactId>iotdb-jdbc</artifactId>
      <version>0.0.1-SNAPSHOT</version>
    </dependency>
</dependencies>
```
